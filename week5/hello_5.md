    - -모든 요소를 컴포넌트 단위로 쪼개어 개발후 하나로 합쳐 페이지 구성

    - 첫 컴포넌트 만들기
        - 자바스크립트의 클래스, 함수 이용해 만들기(함수 선호)
    - 함수 컴포넌트 만들기
        - ex) import “./App.css”;
        - 항상 대문자로 시작
    - 컴포넌트를 페이지에 렌더링하기
        - <App/>(부모)→<header/>,<Body/>,<Footer/>(자식)
    - 컴포넌트별로 파일 분리

- 2.JSX
    - jsx: 컴포넌트를 만들 때 사용하는 문법
    - 표현식
        - 산술표현식: 숫자로 표현
        - 문자열 표현식: 문자열로 평가
        - 논리 표현식: 참,거짓
    - 사용할 수 없는 값
        - 문법으로 저장→ 아무것도 나타나지 않으며 오류 발생
        - “Object are not valid as a React Child”
    - 문법 규칙
        - 닫힘 규칙
        - 최상위 태그 규칙
    - 조건부 렌더링
        - 삼항 연산자를 활용한 조건부 렌더링
        - 조건문을 이용한 조건부 렌더링
    - jsx 스타일링
        - 인라인 스타일링
        - 스타일 파일 분리
- 3.컴포넌트에 값 전달하기
    - Props: 부모가 자식 컴포넌트에 단일 객체로 값 전달 가능
    - 주의
        - 하나의 값 전달
        - 여러개의 값 전달
        - 구조 분해 할당으로 여러 개의 값 사용
        - 스프레드 연산자로 여러개의 값 전달
        - 기본값 설정
        - 컴포넌트 전달
- 4.이벤트 처리
    - 이벤트 핸들링 : 특정 코드 동작
